# Map Brief: Map 042 - Cardboard Woods

**Theme:** A dense forest crafted entirely from layered construction paper and cardboard, giving it a surreal, cut-out appearance.

**Atmosphere:** Whimsical, slightly unsettling, with a distinct visual texture of paper-craft materials. The background music should have a soft, "crinkly" paper-inspired sound.

**Key Elements:**
- **Glitch Circle:** Positioned in a central clearing. Interacting with it triggers a transformation that changes the actor's sprite to a Paper Plane (State ID 82: Airborne).
- **Chasm (Region 10):** A wide gap in the cardboard ground, impassable in normal form. While in Paper Plane (Airborne) form, the player can glide over this region.
- **Encounter Zones:**
    - ID 121: Goomba-Equiv (Flipped sprite).
    - ID 122: Koopa-Equiv (Shell-toss mechanics).
    - ID 130: Hooktail-Equiv (Boss).

**Builder Instructions:**
- Use the Paper-Craft tileset.
- Ensure the layering effect is maintained with transparent paper-texture overlays.
- Set the passability of Region 10 to impassable unless State 82 is active.
